# Dados Externos

Bases externas recebidas junto com o material dos colegas.

## Estrutura

- `cnes/`: arquivos CSV compactados do CNES para 2024 e 2025.
- `municipios/`: shapefile compactado de municipios do Brasil.

## Uso

Nao entram na primeira versao da modelagem. Devem ser avaliados depois como possivel enriquecimento de contexto geografico ou saude/regiao.
